# study_notes
SUBU CENG Study Notes

## Repo Structure
Use the same folder structure for each course:

- `resources`         PDFs/slides/notes shared by the instructor
  - `images`          Images used in READMEs and documentation
- `notes`             Notes
- `past_exam`         Past exam questions
- `assignments`       Assignments and submissions

You can copy the `_course_template` folder as a ready-made skeleton.
